# admin-leyu-resource

该仓库用于归档和发布多个独立的 HTML 页面，内容涵盖管理、资源展示、信息聚合等常见用途。所有页面均为静态 HTML，不依赖特定域名或后端服务，便于快速浏览、测试或嵌入使用。

## 目录说明

```
admin-leyu-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── docs/           # 文档或说明性文件（可选）
└── README.md       # 本文件
```

- `pages/` 目录下的每个 HTML 文件为一个独立页面，可直接在浏览器中打开。
- `assets/` 目录按需组织，用于存放页面所需的公共或私有资源。

## 页面归档说明

- 所有页面均为纯前端实现，无后端依赖。
- 页面内容不针对任何特定网站或域名设计，仅作为通用功能演示或信息展示。
- 部分页面可能包含示例数据，均用于演示目的，无实际业务含义。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 下的 HTML 文件。
3. 如需部署到服务器，将整个仓库上传至任意静态文件服务器即可。
